[h1] Tooltip Icon Patch [/h1]
An Addon to [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3582340033]Tooltip Icon[/url]

[b]Tooltip Icon[/b] is a great mod, but it only supports vanilla damage types. All other modded damage types use the default melee class icon. To make it better fit in with other mods, we create this patch to add compatibility with some other mods.
